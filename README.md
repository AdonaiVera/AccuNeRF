## Install

Clone repo with submodules:
```
git clone https://github.com/AdonaiVera/AccuNeRF.git --recurse-submodules
git submodule update --init --recursive
```

Create a virtualenv 
```
conda env create --file environment.yml
conda activate accunerf
```
